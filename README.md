# MarketLens 📊

Stock & Crypto Analytics Dashboard built with Python.

## Features
- Real-time stock & crypto data (yfinance)
- Interactive price charts (Plotly)
- Technical indicators (MA20, MA50, RSI, Volatility)
- Performance metrics
- Clean dashboard UI (Streamlit)

## Tech Stack
- Python
- Streamlit
- yfinance
- pandas / numpy
- Plotly

streamlit run app.py
